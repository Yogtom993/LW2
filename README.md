# Лабораторна робота №2

## Тема
CSS Grid Layout, БЕМ-методологія та медіазапити.

## Опис
У роботі сайт TechStore з ЛР1 перероблено за допомогою CSS Grid Layout. Для стилів використано БЕМ-підхід, додано адаптивність, окрему Grid-сторінку, fixed-елемент, sandwich-меню та галерею з CSS-анімацією.
