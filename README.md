# Лабораторна робота №2

## Тема

CSS Grid Layout. Медіазапити.

## Назва проєкту

**TechStore — каталог продукції компанії**

## Мета роботи

Ознайомитися з можливостями CSS Grid Layout, принципами БЕМ-розмітки та медіазапитами. Навчитися створювати адаптивний дизайн вебсторінок і правильно організовувати структуру CSS-коду.

## Опис проєкту

У межах лабораторної роботи було перероблено сайт **TechStore**, створений у лабораторній роботі №1.

Основна структура сайту залишилася такою самою, але компонування сторінок було адаптовано за допомогою CSS Grid Layout, БЕМ-методології та медіазапитів.

## Виконані завдання за методичкою

### Блок №1

#### Завдання №1

**Розташуйте ваш контент на сторінках з лабораторної роботи №1 згідно з концепцією CSS Grid Layout. Завдання повинно бути виконане без зміни структури HTML-документа.**

Контент сторінок сайту TechStore було розташовано за допомогою CSS Grid Layout.
У файлі `style.css` для основних контейнерів використано властивості:

* `display: grid`;
* `grid-template-columns`;
* `grid-template-areas`;
* `gap`.

Структура HTML-документів залишилася без суттєвих змін, а розташування блоків виконано засобами CSS.

#### Завдання №2

**Модифікувати CSS-файл згідно з БЕМ-методологією.**

CSS-класи було оформлено за БЕМ-методологією.
Для елементів сайту використано структуру назв типу:

```text
block
block__element
block__element--modifier
```

Наприклад:

* `header`;
* `header__title`;
* `nav`;
* `nav__link`;
* `card`;
* `card__title`;
* `product`;
* `product__image`.

### Блок №2

#### Завдання №1

**Реалізувати додаткову сторінку зі структурою, що надасть викладач, використовуючи CSS Grid Layout.**

Було створено додаткову сторінку `grid.html`, на якій реалізовано сіткову структуру за допомогою CSS Grid Layout.

#### Завдання №2

**Виконати умови щодо відображення сторінки з попереднього завдання на мобільних пристроях з різною орієнтацією екрана.**

Для адаптації сторінки використано медіазапити `@media`.
При зменшенні ширини екрана блоки перебудовуються в одну колонку, а навігаційне меню змінює своє розташування.

#### Завдання №3

**У будь-якому кутку сторінки розмістити fixed-елемент, який деформується при наведенні мишкою.**

На сторінці додано фіксований елемент, розміщений у кутку екрана.
Для нього використано `position: fixed`, а при наведенні мишкою застосовано зміну форми через CSS.

### Блок №3

#### Завдання №1

**Реалізувати меню типу “sandwich” для екранів шириною 320px.**

Для малих екранів реалізовано компактне меню типу “sandwich”.
Воно використовується для зручної навігації на мобільних пристроях.

#### Завдання №2

**Створити галерею з анімацією зображень.**

На сторінці каталогу створено галерею зображень.
При наведенні мишкою зображення змінює масштаб і вигляд за допомогою CSS-анімації.

## Структура проєкту

```text
LR2_TechStore/
├── index.html
├── catalog.html
├── contacts.html
├── grid.html
├── style.css
├── images/
└── README.md
```

## Опис файлів

* `index.html` — головна сторінка сайту TechStore з Grid-структурою;
* `catalog.html` — сторінка каталогу товарів і галереї;
* `contacts.html` — сторінка контактів і форми;
* `grid.html` — додаткова сторінка для демонстрації CSS Grid Layout;
* `style.css` — файл стилів із CSS Grid Layout, БЕМ-класами та медіазапитами;
* `images/` — папка із зображеннями товарів;
* `README.md` — опис проєкту.

## Як запустити проєкт

1. Завантажити або клонувати репозиторій.
2. Відкрити папку проєкту.
3. Запустити файл `index.html` у браузері.
4. Для переходу між сторінками використовувати навігаційне меню.
